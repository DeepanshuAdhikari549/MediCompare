# 🏥 MediCompare

🚀 **Live Demo:** https://medi-compare-omega.vercel.app  

---

## 📌 Overview

**MediCompare** is a full-stack healthcare platform that helps users find and compare hospitals, lab tests, and medical services at the lowest prices nearby.

It provides **real-time comparison, booking system, AI assistance, and admin analytics** — all in one place.

---

## ✨ Key Features

- 🔍 **Smart Search** — Find hospitals, labs, clinics by service/location  
- 💰 **Price Comparison** — Compare MRI, CT Scan, Blood Tests, etc.  
- 📍 **Geolocation Support** — Nearby hospitals (fallback to Dehradun)  
- 🤖 **AI Assistant** — Gemini-powered medical chatbot  
- 📅 **Appointment Booking** — Slot-based scheduling + home sample collection  
- 💳 **Payment Simulation** — UPI/Card + Invoice PDF generation  
- 👤 **Role-Based Auth** — Patient | Hospital | Admin  
- ⭐ **Ratings & Reviews** — Verified user feedback  
- 📊 **Admin Dashboard** — Analytics + hospital verification  
- 🌙 **Dark Mode UI** — Fully responsive modern design
